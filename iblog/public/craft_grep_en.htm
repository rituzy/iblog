Grep tool for Windows.

Some time before I had to find some strings in logs. If we talk about Windows there are some apps to help us with this task: TotalCommander's lister or Notepad++. But what if you want to have all the necessary strings and only them? Of cause you can install unixtools and use grep command or Windows' findstr from cmd. I wanted to create something useful after I had read thick Java book of Herbert Schildt as an exercise. In the end I had a small app which finds strings containing search string in the file of all files in the folder. Project has beed created in NetBeans.
Important points:
  • I developed a multithread version but after tests on different machines left it as is
Features:
  • find both Russian and Englis strings
  • is able to find in the file or in all files of the folder
  • is able to find regular expressions
