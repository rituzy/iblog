Вернуться в блог

GUI грепалка для Windows.

Несколько раз уже возникала необходимость по работе грепать логи, причем с кирилицей. В Windows есть уже примерно похожие вещи: листер TotalCommander'а или Notepad++. Однако, хочется видеть только все нужные строки в результате поиска. Здесь поможет установка unixtools(можем использовать старый добрый grep) или использование findstr из командной строки(что-то надо доставить дополнительно, чтобы доступна была команда) Мне ж захотелось в рамках упражнения после прочтения талмуда по Java Шилдта совместить приятное с полезным. В итоге получилась скромная програмка, которая ищет вхождение искомой строки в файлах содержащих текст, на который укажут(или во всех файлах в директории). Проект сделан в NetBeans.
Особенности:
  • была у меня версия с использованием многопоточности, но после экспериментов вернул однопоточную версию
Фишки:
  • ищет как русские так и английские строки
  • может делать обход по всем файлам в папке
  • можно искать регулярное выражение
